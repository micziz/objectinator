# objectinator

Transform an array into an object

## Installation

```shell
npm i objectinator
```

or if you use yarn or pnpm

```sh
yarn add objectinator
```

pnpm

```sh
pnpm add objectinator
```

## Usage

```js
import { transformIntoObject } from 'objectinator';

let arr = ["Hi", "Hello"]

const obj = transformIntoObject(arr)
console.log(obj)
```
